### Installing locally

1. Make sure you have Node.js to run the application
2. Clone this repo
> HTTPS: `https://github.com/micheqn3/redux-store.git` <br>
> SSH: `git@github.com:micheqn3/redux-store.git`
3. Install the NPM packages
> npm install
4. Run the application
> npm run develop
